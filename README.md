# CareerFit AI

> 취업·공모전 데이터 기반 맞춤형 AI 포트폴리오 코치



## 프로젝트 개요

금융권 데이터 분석 직무(카드사, 은행 등) 채용은 단순히 코딩 테스트나 자격증만으로 통과할 수 없습니다. 현업에서는 수만 개의 행에 달하는 Raw Data를 정제하고, 엑셀을 넘어 SAS, R 등의 분석 툴을 활용해 유의미한 비즈니스 인사이트를 도출하는 실전 역량을 요구합니다.

CareerFit AI는 파편화된 금융권 직무 정보를 분석하고, 지원자의 과거 데이터 다루기(전처리, 마케팅 분석 등) 경험을 금융 도메인에 어떻게 연결할지 코칭해 주는 지능형 커리어 매칭 플랫폼입니다.

● 문제상황
데이터 분석가나 디지털 뱅커를 꿈꾸는 취업 준비생들은 '금융'과 '데이터'라는 두 마리 토끼를 잡아야 하지만, 이를 둘러싼 정보 비대칭이 심각합니다.

    카드사 (데이터 분석): 마케팅 분석, 신용평가(CSS), CRM 등 세부 직무가 어떻게 다르고, 각 부서에서 구체적으로 어떤 툴과 데이터를 다루는지 알기 어렵습니다.

    은행 (일반/디지털 뱅커): 전통적인 금융 업무와 디지털/데이터 업무가 어떻게 융합되고 있는지, 현장에서 요구하는 '데이터 리터러시'의 수준이 어느 정도인지 파악하기 힘듭니다.

    경험의 연결 부족: 소비재나 타 산업군에서 마케팅 데이터를 다뤄본 경험, 통계학적 지식 등을 금융권 자소서와 면접에 어떻게 녹여내야 할지 막막해합니다.

● 기존 방식의 한계

    일반 채용 포털: 직무가 '데이터 분석' 혹은 '디지털/IT'로 뭉뚱그려져 있어, 실제 내 역량(통계 모델링, 데이터 전처리 등)과 맞는 자리인지 알 수 없습니다.

    일반 챗봇 (ChatGPT 등): 그럴듯하게 답변하지만, 실제 카드사나 은행의 최신 채용 트렌드나 현업 부서의 정확한 업무 내용을 반영하지 못하고 원론적인 이야기만 반복합니다.
